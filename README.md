# Create Gantt chart in draw.io from Notion export

1. Open Notion page with cleaned and finished Gantt Chart
2. Top right corner 'more' -> 'Export'
3. Leave default options

<img width="285" height="217" alt="image" src="https://github.com/user-attachments/assets/8b79d1d3-0e3d-47f8-8706-d97765204c36" />

4. Clone this repo
5. Extract downloaded zip from 3. into the repo folder
6. In the code change csv_path to the .csv file name - **MAKE SURE IT IS THE ONE ENDING WITH _all**
7. Run the code and upload you .drawio file to google drive, then open it with draw.io
8. Enjoy
